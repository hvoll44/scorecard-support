ScoreCard App Support

Welcome to the support page for the ScoreCard app. Here, you will find information on how to use the app, troubleshoot issues, and get help.

Frequently Asked Questions

  • How do I sync scores in real-time?
    Someone must create a new game and click the share icon. Once it displays a 6-digit code others can click Join Game from the main menu. Enter the code and enjoy. Scores will be updated in real-time for all participants.
  • Where is my data stored?
    Data is stored locally on your device and is not shared with any external services. Shared games are temporarily stored in the cloud to enable syncing scores with others. Games that are not shared only store data on your local device.

Troubleshooting

If you encounter any issues with the app, try the following:

  • Ensure your app is up to date.
  • Check your internet connection for syncing issues.
  • If shared games are not syncing in real-time, try turning the screen off and on.
  • Restart the app if it becomes unresponsive.
